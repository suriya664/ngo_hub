# NGO Website Template

A responsive NGO website template built with HTML5, CSS3, JavaScript, jQuery, and Bootstrap 5.

## Features

- Fully responsive design
- Modern, clean UI with NGO-appropriate color scheme
- Hero section with statistics
- Campaign showcase
- Testimonials section
- How It Works section
- Call-to-action sections
- Mobile-friendly navigation
- Smooth scrolling
- Dynamic content loading

## Color Palette

- Primary Orange: #ff6b35
- Secondary Blue: #0466c8
- Accent Cyan: #00b4d8
- Success Green: #28a745
- Warning Yellow: #ffc107
- Danger Red: #dc3545
- Light Background: #f8f9fa

## Typography

- Font Family: Inter / Poppins (Google Fonts)
- All paragraph text: text-align: left
- Consistent spacing and alignment

## File Structure

```
ngo-template/
├── index.html              # Main homepage
├── assets/
│   ├── css/
│   │   └── style.css      # Main stylesheet
│   ├── js/
│   │   └── main.js        # Main JavaScript
│   └── img/
│       └── logo.svg        # Logo file
├── partials/
│   ├── header.html         # Header component
│   ├── footer.html         # Footer component
│   └── modals.html         # Modals component
└── pages/                  # Additional pages (to be created)
```

## Getting Started

1. Open `index.html` in a web browser
2. For best results, use a local web server (not file:// protocol)
3. The template uses Fetch API to load partials, so it requires a web server
